AWS Projects

A collection of projects I have completed or am working on. Page will be updated weekly with completed projects. If you are interested in using any of the templates below or would like me to create a custom one for you below please contact me on Twitter or LinkedIn. I do not charge for my services at this time so in lieu of any payments please donate whatever you think my work is worth to a charity of your choice.


1. Personal Website Template

Template of my personal static website hosted on S3 using a custom domain name with Route53. A CloudFront distribution sits in front of the S3 bucket to serve static content and a CloudFormation stack is saved to S3 as a template in case of the need for disaster recovery.


2. Web Server Template

Template of a web server inside of a simple VPC setup. The VPC consists of an EC2 instance, a public subnet with a security group to allow for HTTP/SSH access, a route table attached to determine where network traffic is to be routed, and an internet gateway attached to the VPC to allow outside internet access from the VPC.
